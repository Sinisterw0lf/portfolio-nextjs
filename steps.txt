NextJS : 

1- Installer contentful = npm i contentful
2- Connexion à l'API contentful dans settings->API keys->Add API key
3- Creer un fichier .env et verifier dans gitignor que c'est ignoré
4- Dans .env creer les variables d'environnement en uppercase prefixé par NEXT_PUBLIC
5- Coller les variables d'environnement récupérer dans contentful
6- Dupliquer .env.local et renommer en.env-exemple pour aider la personne qui reprendra le projet

Description->Rich-text : npm install @contentful/rich-text-react-renderer
